Classical Ciphers

  • Caesar Encryption
  • This allows you to select a rotation number as a key and will rotate the each letter by your chosen key. For example with a key of 1 A>B, B>C and so on.
  • Reverse Encryption
  • This flips the alphabet from Z-A and replaces each letter with the corresponding letter in the flipped alphabet. For example ABC>ZYX.
  • Morse Code
  • Morse code is a method of sending text messages by keying in a series of electronic pulses, usually represented as a short pulse (called a "dot") and a long pulse (a "dash").
